# C# Practical Interview Questions with Solutions (One View)

## 1. Reverse a String Without Using Built-in Function

```csharp
string input = "Pradeep";
char[] arr = input.ToCharArray();

int start = 0;
int end = arr.Length - 1;

while (start < end)
{
    char temp = arr[start];
    arr[start] = arr[end];
    arr[end] = temp;

    start++;
    end--;
}

Console.WriteLine(new string(arr));
```

---

## 2. Check Palindrome String

```csharp
string input = "madam";
string reversed = new string(input.Reverse().ToArray());

if (input == reversed)
    Console.WriteLine("Palindrome");
else
    Console.WriteLine("Not Palindrome");
```

## 3. Find Duplicate Characters in String

```csharp
string input = "programming";

var duplicates = input.GroupBy(x => x)
                      .Where(g => g.Count() > 1)
                      .Select(g => g.Key);

foreach (var d in duplicates)
    Console.WriteLine(d);
```

---

## 4. Count Character Frequency

```csharp
string input = "hello";

var result = input.GroupBy(c => c)
                  .Select(g => new { Character = g.Key, Count = g.Count() });

foreach (var item in result)
    Console.WriteLine(item.Character + " : " + item.Count);
```

---

## 5. Find Second Largest Number in Array

```csharp
int[] numbers = { 10, 20, 5, 8, 30, 25 };

int secondLargest = numbers.Distinct()
                           .OrderByDescending(x => x)
                           .Skip(1)
                           .First();

Console.WriteLine(secondLargest);
```

```csharp
List<int> numbers = new List<int> { 1, 2, 2, 3, 4, 4, 5 };
```

## 8. Fibonacci Series

```csharp
int n = 10;

int a = 0, b = 1;

for (int i = 0; i < n; i++)
{
    Console.Write(a + " ");

    int temp = a + b;
    a = b;
    b = temp;
}
```

---

## 9. Check Prime Number

```csharp
int number = 29;
bool isPrime = true;

for (int i = 2; i <= number / 2; i++)
{
    if (number % i == 0)
    {
        isPrime = false;
        break;
    }
}

Console.WriteLine(isPrime ? "Prime" : "Not Prime");
```

---

## 10. Factorial Using Recursion

```csharp
int Factorial(int n)
{
    if (n <= 1)
        return 1;

    return n * Factorial(n - 1);
}

Console.WriteLine(Factorial(5));
```

## 14. Sort Object List Using LINQ

```csharp
class Employee
{
    public string Name { get; set; }
    public int Salary { get; set; }
}

var employees = new List<Employee>()
{
    new Employee { Name = "A", Salary = 5000 },
    new Employee { Name = "B", Salary = 7000 },
    new Employee { Name = "C", Salary = 6000 }
};

var sorted = employees.OrderByDescending(e => e.Salary);

foreach (var emp in sorted)
    Console.WriteLine(emp.Name);
```


## 16. Implement Singleton Pattern

```csharp
public class Logger
{
    private static Logger instance;

    private Logger() { }

    public static Logger Instance
    {
        get
        {
            if (instance == null)
                instance = new Logger();

            return instance;
        }
    }
}
```

## 17. Parallel API Calls Using Task.WhenAll

```csharp
async Task CallApis()
{
    Task<string> api1 = client.GetStringAsync("url1");
    Task<string> api2 = client.GetStringAsync("url2");

    var results = await Task.WhenAll(api1, api2);

    Console.WriteLine(results[0]);
    Console.WriteLine(results[1]);
}
```

## 19. Implement Stack

```csharp
Stack<int> stack = new Stack<int>();
```

## 20. Binary Search

```csharp
int BinarySearch(int[] arr, int target)
{
    int left = 0;
    int right = arr.Length - 1;

    while (left <= right)
    {
        int mid = (left + right) / 2;

        if (arr[mid] == target)
            return mid;

        if (arr[mid] < target)
            left = mid + 1;
        else
            right = mid - 1;
    }

    return -1;
}
```
